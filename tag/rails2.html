在 Rails 中安裝與設定 Bootstrap

假設有個頁面的部分程式碼如下

%header
  %h3 #{@video.title}
  %span Rating:
  - if @video.rating
    = "#{@video.rating}/5.0"
  - else
    N/A
%p #{@video.description}

%header
  %h3 #{@video.title}
