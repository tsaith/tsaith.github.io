一套免費的 SQLite 資料庫瀏覽器

在軟體開發時，我們有時會使用輕量的 SQLite 資料庫來做測試; 這裡要介紹一套好用又免費的工具 DB Browser for SQLite， 它可以用來幫助使用者方便地建立資料庫和搜尋或編輯其中的資訊。

安裝方式

打開終端機，使用 Homebrew 來進行安裝

brew install sqlitebrowser

使用方法

在命令列直接執行 sqlitebrowser 來啟動 SQLite Brower

sqlitebrowser

在界面的工具列點選 Open Database ，然後選擇 SQLite 資料庫所在的位置， 就能開始存取裡面的資訊
